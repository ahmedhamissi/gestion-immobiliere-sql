# Projet – Base de données relationnelle (Gestion immobilière)

Projet académique réalisé dans le cadre du cours **IFT187** à l’Université de Sherbrooke.  
Conception et implémentation d’une base de données destinée à une compagnie de gestion immobilière.

## Contenu
- Modélisation : MCD et MRD (dossier `modeles/`)
- Scripts SQL : création des tables, insertions et fonctions (dossier `sql/`)

## Technologies
- SQL
- pgAdmin / DataGrip
